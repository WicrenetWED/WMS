LMS Web API 的Token

1. 安装Postman

Postman安装地址

2 . 禁用 SSL 证书验证

在“文件”>“设置”（“常规” 选项卡）中，禁用“SSL 证书验证”。

3. 获取Token的过程

a. 创建新请求。

b. 将 HTTP 方法设置为“GET”。

c. 将请求 URI 设置为 https://139.219.15.177/BPServiceHub/token

​ URL： https://139.219.15.177/BPServiceHub/token

d. 输入参数

参数名 描述 示例
username 用户的域名和名字组合: 域名\用户名 cflogistics\dev1
password 用户的密码 xxxxx

e. 点击Send

f. 输出token
