13 MSDS & QAC文件主数据同步

13.1 接口说明

项目 描述
功能 用于传输SDS 和 QAC文件主数据信息
数据方向 LMS–>WMS
实时性 按需
说明 期初一次性维护，后续WMS按需拉取数据

13.2 请求URL与方式

URL：api/MSDSAndQACFileTransfer

测试地址：http://139.219.15.177/BPServiceHub/api/MSDSAndQACFileTransfer

协议：http

方式：POST

13.3 输入参数

Header 需传参Token

参数 描述 类型 是否必填 备注
Token 授权令牌 String 是

请求Body参数

参数 描述 类型 是否必填 备注
ItemId 物料编号 String 是
Type 类型 String 是 MSDS, QAC, All (表示同时获取MSDS和QAC文件)
InventBatchId 批号 String 当Type是QAC 和 All 时 必填
IsVersion 是否仅仅检查版本信息 bool 否 默认值：false

13.4 输出参数

103 处理成功
104 空内容
105 不正确的请求
